Help & Information

Features

  • Assists in searching through popular tourist attractions in the state of Florida
  • Lists all available tourist attractions
  • Filter the list search by attraction name, type, group size, rating, and price!
  • 185 entries to search through!

Using the Search Tool

To use the search tool:

  1. Type in and/or select what to search for in the menu
    • You can select multiple search types at once
  2. Press "Search"
  3. Results will be displayed

Note:

  • Searching with no input displays all locations
  • Name: searches for the attraction name (non-case-sensitive)
  • Type: displays all attractions in the selected category
  • Price: displays all free attractions
  • Group Size: displays attractions with the selected recommended group size
  • Rating: displays attractions with the selected rating, out of 5

Information

  • Created by Michelle McAveety at Freedom High School for the 2021-2022 FBLA Coding and Programming competition
  • All attraction name, address, telephone, and description data was accessed from https://www.florida-backroads-travel.com/florida-tourist-attractions.html
  • Entries were omitted in scenarios where the aforementioned data was missing or inconsistent
  • Data was unaltered except to improve clarity or program functionality
  • Attraction type, group size, rating, and price data were generated for purposes of program search functionality
  • Type was assigned algorithmically where "museum" or "park" was present in attraction name. Remaining entries were manually assigned attraction types
  • Price was algorithmically assigned based on attraction description and manually checked for accuracy
  • Group size was algorithmically assigned based on attraction type. In instances where attraction type was "other", group size was manually assigned
  • Rating was pseudorandomly assigned to each entry
